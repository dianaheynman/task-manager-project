# Task Manager – אפליקציית ניהול משימות

אפליקציה פשוטה לניהול משימות, שנבנתה במסגרת פרויקט מסכם בקורס React.  
האפליקציה מאפשרת לנהל רשימת משימות יומיומיות בצורה נוחה וברורה.

## מה אפשר לעשות באפליקציה
- להוסיף משימה חדשה
- לסמן משימה כהושלמה או לא הושלמה
- לערוך טקסט של משימה קיימת
- למחוק משימה
- לסנן משימות לפי: הכל / פעילות / הושלמו
- לראות כמה משימות פעילות נשארו
- המשימות נשמרות גם אחרי רענון הדף בעזרת LocalStorage

## טכנולוגיות
- React 
- Vite
- JavaScript
- CSS 
- LocalStorage

## מבנה הקומפוננטות
TaskForm-אחראית על קלט המשתמש והוספת משימות חדשות.

TaskList-מציגה את רשימת המשימות ומעבירה לכל משימה את הנתונים והפעולות

TaskItem-מציגה משימה בודדת כולל checkbox, עריכה ומחיקה.

FilterBar-אחראית על כפתורי הסינון.
